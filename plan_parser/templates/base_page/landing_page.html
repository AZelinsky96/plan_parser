Welcome To Plan Parser!

Introduction:

This application was built to allow users to have an interface to process slcsp`s. An slcsp is a second lower cost silver plan. This application identifies the second lower cost silver plan for desired areas specified in input files.

The process works in the following way:

  1. Upload source files (zip codes, plans, and slcsp)
  2. Validate and Process the files
  3. Download the processed data as a csv.
